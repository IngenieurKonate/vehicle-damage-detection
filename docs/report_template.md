# Template du Rapport Académique

## Structure Recommandée

### 1. Introduction
- Contexte et motivation
- Problématique de recherche
- Objectifs du projet

### 2. Revue de Littérature
- Architectures CNN (VGG, ResNet)
- Détection de dommages véhicules
- Datasets existants

### 3. Méthodologie
- Description des datasets (CarDD, Stanford Cars)
- Prétraitement et augmentation
- Architectures proposées (Model A, Model B)
- Pipeline d'entraînement

### 4. Expérimentations
- Configuration expérimentale
- Hyperparamètres
- Environnement technique

### 5. Résultats
- Courbes d'apprentissage
- Métriques de performance
- Comparaison des modèles
- Matrices de confusion

### 6. Discussion
- Analyse des résultats
- Validation des hypothèses
- Analyse des erreurs
- Limites du projet

### 7. Conclusion
- Synthèse des contributions
- Perspectives futures

### Annexes
- Code source important
- Détails des architectures
- Résultats complémentaires
